# Course GitHub Skills: GitHub Pages
With **GitHub Pages**, you can host and publish *static* content from a GitHub repository. This course introduces the main types of content you can create and deploy with GitHub Pages:
1. Project blogs
2. Documentation
3. Resumes
4. Portfolios
5. Other static content
